#Llibertas tidsskriftet
PDF data extraction.


# Developer Guide

This project uses the `uv` package manager (see <https://docs.astral.sh/uv/>).

- Create a virtual environment:
  ```
  uv venv create
  ```

- Activate it:
  ```
  .venv\Scripts\activate
  ```

- Install dependencies:
  ```
  uv install
  ```
